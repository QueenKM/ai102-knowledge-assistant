# BrewMaster Mini Error Code Guide

## Error E12
Error E12 means low water flow during brewing. Run a full descale cycle, rinse the water tank, and confirm that the intake tube is seated correctly.

## Error E21
Error E21 means the brew chamber is blocked. Remove the capsule, power cycle the machine, and clean the chamber before retrying.

## When to replace the unit
If the same error returns after two full troubleshooting cycles and the device is still within warranty, create a warranty replacement case instead of scheduling a repair.
